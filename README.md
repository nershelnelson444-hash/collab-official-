# co//ab. Digital Agency

High-end, strict minimalist digital agency portfolio built with **Next.js (App Router)**, **Tailwind CSS v4**, and **Framer Motion**.

## Overview
This repository contains the single-page scrollable frontend architecture for co//ab., designed with a strict monochrome aesthetic (`#000000`, `#FFFFFF`, and muted gray borders). It features smooth snap-scrolling, infinite tech marquees, and a bento grid layout for case studies.

## Tech Stack
- **Framework:** Next.js 16.x (App Router)
- **Styling:** Tailwind CSS v4 (Custom @theme variables)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/bhaviksaj/collab.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design System Constraints
This project adheres to strict design tokens:
- **Typography:** Inter (Sans/Heading), Geist (Mono/Label).
- **Colors:** `#000000` Background, `#FFFFFF` Primary Text, `#A3A3A3` Muted Text.
- **Borders:** `#222222` Card Borders, `#333333` Input Borders (Hair-thin 1px solid).
- **Interactions:** Subtle `1.02x` scale on hover for cards. Form inputs use border-bottom expansion on focus.
